# BiliHDPager

哔哩哔哩 HD 分 P 视频修复模块。

[English](README.en.md)

## 功能

这是一个 LSPosed 模块，用于修复旧版哔哩哔哩 HD 客户端里一类分 P 视频无法正常切换的问题。

部分视频看起来是普通分 P，但客户端内部会按合集/剧集视频处理，导致点击分 P 没反应，或者页面状态没有正确切换。本模块会在播放页修正这类视频的分 P 跳转逻辑。

## 适用范围

- 目标应用：哔哩哔哩 HD
- 包名：`tv.danmaku.bilibilihd`
- 需要 LSPosed 或兼容框架
- 发布版 APK 仅支持 `arm64-v8a` 设备
- 模块没有桌面图标，也没有设置界面

### 支持的客户端版本

| 哔哩哔哩 HD 版本 | 状态 | 说明 |
| --- | --- | --- |
| `1.38.0` | 已测试 | 分 P 播放页、全屏选集、缓存选集已验证 |
| `1.47.0` | 已测试 | 分 P 播放页、全屏选集、缓存选集已验证 |
| 其他版本 | 尝试兼容 | 模块会尝试自动匹配混淆后的播放源、缓存和选集相关类，但不保证可用 |

不同 HD 客户端版本的混淆类名和方法名可能不同。未列出的版本如果没有生效，请提供 LSPosed 日志和客户端版本号。

本模块只处理分 P 切换问题，不会解锁会员内容、地区限制、清晰度限制，也不会修改账号权限。

## 下载哪个 APK

| 你的 LSPosed 版本 | 推荐 APK |
| --- | --- |
| v2.1.0 (7769) 或更新版本 | `BiliHDPager-0.1.0-modern-api102.apk` |
| 早于 v2.1.0 (7769) | `BiliHDPager-0.1.0-legacy.apk` |

不要同时安装两个版本。如果不确定自己的 LSPosed 版本，请先在 LSPosed 设置里查看版本号。

## 使用方法

1. 安装对应的 APK。
2. 在 LSPosed 中启用模块。
3. 作用域选择哔哩哔哩 HD。
4. 强行停止并重新打开哔哩哔哩 HD。
5. 打开之前无法切换后几 P 的视频测试。

## 反馈问题

反馈时请尽量提供：

- 哔哩哔哩 HD 版本号
- LSPosed 版本号
- 安装的是 `legacy` 还是 `modern-api102`
- 出问题的视频链接或 BV 号
- LSPosed 导出的日志

如果需要排查具体兼容问题，可以临时安装 debug 包。debug 包会把更详细的诊断信息写入 LSPosed 日志，日志 tag 为 `BiliHDPager`。
